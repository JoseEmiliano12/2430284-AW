Práctica Frontend

¿1. Qué es JavaScript y cuáles son sus principales usos en la web?

Es un lenguaje de programación que le agrega interactividad a la página funcionando junto con HTML y CSS para poder crear páginas web dinámicas y funcionales.

2. ¿En qué año y con qué propósito se creó JavaScript?

JavaScript fue creado por Brendan Eich en el año 1995 con el propósito de poder añadir interactividad y funcionalidad a las páginas web.

3. ¿Por qué JavaScript no es lo mismo que Java?

Porque JavaScript se diseñó para la interactividad en las páginas web y Java es un lenguaje de programación de propósito general.

4. ¿Cómo se complementa JavaScript con HTML y CSS?

Se complementan ya que HTML proporciona la estructura de la página, CSS aplica el diseño y el estilo, y JavaScript añade interactividad y funcionalidad dinámica.

5. ¿Cuál es la sintaxis básica de JavaScript y cómo se escriben comentarios?

La sintaxis básica de JavaScript incluye palabras clave, identificadores, operadores y estructuras de control. Los comentarios se pueden escribir utilizando // para comentarios de una sola línea y /* */ para comentarios de varias líneas.

6. ¿Qué diferencias hay entre var, let y const?

var: Tiene alcance de función, se puede redeclarar y reasignar
let: Tiene alcance de bloque, se puede reasignar pero no redeclarar
const: Tiene alcance de bloque, no se puede reasignar ni redeclarar

7. ¿Cuáles son los tipos de datos primitivos en JavaScript? Da un ejemplo de cada uno.

Existen 7 tipos de datos primitivos en JavaScript:
- String: Representa texto. Ejemplo: let nombre = "Juan";
- Number: Representa enteros como decimales. Ejemplo: let edad = 25;
- Boolean: Solo puede tener dos valores true o false. Ejemplo: let esMayor = true;
- Undefined: Variable que no ha sido asignada. Ejemplo: let x;
- Null: Ausencia intencional de valor. Ejemplo: let y = null;
- Symbol: Valor único e inmutable. Ejemplo: let id = Symbol("id");
- BigInt: Números enteros muy grandes. Ejemplo: let numeroGrande = 123n;

8. ¿Qué diferencia hay entre un array y un objeto en JavaScript?

Los arrays se usan cuando se necesita una lista ordenada de elementos, mientras que los objetos se usan para representar algo con propiedades con nombre.

9. ¿Qué son y cómo se usan los operadores aritméticos, lógicos y relacionales?

Los operadores son símbolos que permiten realizar operaciones sobre valores o variables. Existen varios tipos como el Aritmetico: para calculos matematicos, Logicos: Para tomar decisiones basadas en condiciones y la relacionales: para poder comparar valores.

10. ¿Cómo funcionan las sentencias condicionales if, else y switch?

El if, else if y else se usa cuando necesitas evaluar condiciones simples o multiples y el Switch cuando se compara un valor contra muchas opciones diferentes.

11. ¿Cuáles son las diferencias entre los bucles for, while y do...while?

- Bucle for: Se usa cuando se conoce cuantas veces se va a repetir un bloque de codigo
- Bucle while: Se usa cuando quieres repetir un bloque mientras una condicion sea verdadeda
- Bucle do.....while: Es muy similar al while, pero la condicion se evalua despues de ejecutar el bloque

12. ¿Para qué sirven las palabras clave break y continue?

Break: Sirve para detener la ejecucion del bucle completamente
Continue: Sirve para saltat la iteracion actual del bucle y pasar directamente a la siguiente

13. ¿Cómo se declara una función y cómo se llama en JavaScript?

Declarar una funcion: Para declarar uan funcion se usa function y define un bloque de codigo
Llamar una funcion: Para llamar una funcion se usa su nombre seguido de ()

14. ¿Qué significa que una función tenga parámetros y un valor de retorno?

Las funciones que tienen parametros y valores de retorno sirven por que permiten escribir codigo reutilizable, flexible y organizado con los parametros una funcion puede trabajar con diferentes datos sin necesidad de reescribirse. y con el valor de retorno, puede devolver resultados que puedan ser usados en otras partes del programa como calculos condiciones o asignaciones

15. ¿Qué son las funciones flecha (=>) y qué ventajas tienen?

Las funciones flechita (=>) son una forma mas moderna, limpiza de escribir funciones. son ideales para las tareas pequeñas y ayudan a que sea mas legible y eficiente

16. ¿Qué es el scope en JavaScript y cómo afecta a las variables?

El scope en JavaScript determina donde una variable es accesible dentro del codigo.Puede ser global, de función o de bloque, y afecta directamente cómo y dónde puedes usar tus variables.

17. ¿Cómo se declara un objeto y cómo se accede a sus propiedades?

Un Objeto de declara utilizando llaves {} y contiene pares de clave-valor (propiedad y su valor). se puede acceder alas propiedades de dos formas: notacion de punto (.) o notacion de corchetes ([])

18. ¿Qué diferencia hay entre una propiedad y un método en un objeto?

En javaScript los objetos son estructuras claves que permiten agrupan datos las propiedades son las que representan los datos como nombre, edad y los metodos son funciones asociadas al objeto que definen su comportamiento como realizar una accion

19. Menciona al menos 3 métodos básicos de los arrays y explica cómo funcionan.

push(): Agrega uno o mas elementos al final del array
pop(): Elimina el ultimo elemento del array y lo devuelve
shift(): Elimina el primer elemento del array y lo devuelve

20. ¿Qué es el DOM y por qué es importante en el uso de JavaScript en páginas web?

El DOM (Document Object Model) nos ayuda por va de la mano con el html sin el javaScript no podria ver ni cambiar lo que aparece la pagina web

21. ¿Qué es un evento en JavaScript? Menciona al menos 3 ejemplos comunes.

Un evento en JavaScript es una acción o suceso que ocurre en el navegador, como un clic del mouse, una pulsación de tecla o la carga de una página. Ejemplos comunes de eventos son:
- click: Se activa cuando el usuario hace clic en un elemento.
- keydown: Se activa cuando el usuario presiona una tecla.
- load: Se activa cuando una página o un recurso se ha cargado completamente.

22. ¿Cuál es la diferencia entre alert, prompt y console.log?

alert(): solo muestra un mensaje para avisos rapidos
prompt(): muestra un mensaje y espera que el usuario ingrese un valor
console.log(): muestra mensajes en la consola del navegador principalmente para depuracion y desarrollo
